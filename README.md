## 🌟 Project Overview
This repository contains a dynamic **Power BI project** (`power bi project on woo.pbix`). Along with the dashboard file, it also includes the source data (`woo commerce Sales data.xlsx`). This Dashboard is designed to provide actionable insights and performance tracking for "Woo" (a WooCommerce e-commerce store). By leveraging raw sales data, the dashboard delivers a deeply interactive experience, highlighting key performance indicators and sales trends.

---

## 📊 Dashboard Structure

The Power BI report is structured strategically across a primary dashboard page and a supplementary tracking page:

### 1. 📈 Main Sales Dashboard (`Woo Sales Dashboard`)
* **Resolution:** 1280x720 (Standard 16:9 HD Display)*
This is the core dashboard page bringing together a comprehensive suite of **26 visual elements**. It provides an Executive Summary of sales performance with high detail.

**Key Visuals Include:**
- **Interactive Controls:** `3 Slicers` for real-time slicing of data by date ranges, product categories, or regions.
- **Key Performance Indicators (KPIs):** Instant metrics via Card visuals (Total Sales, Total Profit, etc.).
- **Geographical Data:** Map visualization tracking performance across distinct locations.
- **Trend Analysis:** Area chart & Column charts to track revenue flows and spot behavioral differences over time.
- **Categorical Breakdowns:** Clustered Bar Charts, Donut Charts, and Pie Charts comparing segments easily.

### 2. 🗺️ Regional Mapping
* **Resolution:** 1280x720 (Standard)*
A supplementary page designed specifically for intensive geographical insights:
- **Filled Map (Choropleth):** Maps sales densities and revenue heatmaps, providing advanced geographical visualization to highlight top-performing regions.

## 🛠️ Setup & Usage Instructions

To explore the dashboard yourself, follow these instructions:

1. **Prerequisites:** Ensure you have **[Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/)** installed on your machine.
2. **Open the Project:** Launch `power bi project on woo.pbix`
3. **Data Refresh:** Click **Refresh** in the Power BI ribbon. Ensure the dashboard connects to your local instance of `woo commerce Sales data.xlsx` (you may need to update the data source settings to point to your local path).
4. **Interact:** Click on different elements in the donut charts, maps, and slicers to see how visuals cross-filter dynamically!
